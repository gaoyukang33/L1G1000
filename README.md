## 丰富开源的数据处理工具 ##           
Miner U 提供一站式开源高质量数据提取工具，支持多格式(PDF/网页/电子书)，智能萃取，生成高质量预训练/微调语料。    
LabelLLM 专业致力于 LLM 对话标注，通过灵活多变的工具配置与多种数据模态的广泛兼容，为大模型量身打造高质量的标注数据。     
Label U 是一款轻量级开源标注工具，自由组合多样工具，无缝兼容多格式数据，同时支持载入预标注，加速数据标注效率。
